## DevCard Landing Page

Landing page responsiva desenvolvida com foco em construção de interfaces modernas, escaláveis e com comportamento interativo controlado.

O projeto simula um produto digital (SaaS), com estrutura pensada para evolução, manutenção e reutilização de código.

---

## Deploy

https://sibelecaetano.github.io/devcard-landing/

---

## Contexto

Este projeto foi desenvolvido como parte da minha transição para desenvolvimento frontend, com foco em sair do nível conceitual e construir interfaces com comportamento real de produto.

A proposta foi implementar uma landing page completa sem dependência de frameworks, priorizando domínio dos fundamentos.

---

## Principais entregas

- Layout responsivo com abordagem mobile-first
- Menu hambúrguer com controle de estado via JavaScript
- Estrutura semântica e organizada para escalabilidade
- Uso combinado de CSS Grid e Flexbox
- Sistema de estilos baseado em variáveis (design consistente)
- Separação clara entre estrutura, estilo e comportamento

---

## Decisões técnicas

- Utilização de `classList` para controle de estado da interface, evitando manipulação direta de estilos
- Uso de `aria-expanded` para refletir o estado do menu e melhorar acessibilidade
- CSS Grid aplicado para layout estrutural e Flexbox para alinhamentos internos
- Organização do CSS por seções, facilitando leitura e manutenção
- Implementação sem frameworks para reforçar domínio de base

---

## Desafios enfrentados

- Controle de visibilidade do menu mobile sem conflito entre CSS e JavaScript
- Transição de Flexbox para Grid no menu sem quebrar o comportamento
- Gerenciamento de estados visuais (aberto/fechado) de forma consistente
- Evitar sobreposição e conflitos de regras CSS em diferentes breakpoints

---

- Trabalhar responsividade considerando interação (não só layout)
